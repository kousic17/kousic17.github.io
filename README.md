# Kousic Srinivasan — Portfolio (Static)

Plain HTML, CSS, and JavaScript. No build step, no framework.

## Run locally
Just open `index.html` in a browser. Or serve with any static server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy
Upload these three files to any static host (Netlify drop, Vercel, GitHub Pages, Cloudflare Pages, your own server):

- `index.html`
- `styles.css`
- `script.js`

## Edit content
All copy lives in `script.js` (projects, lab projects, services, process, changelog) and `index.html` (hero, contact). Update the arrays at the top of `script.js` to add or change case studies.

## Add your resume
Drop `Resume-Kousic-Srinivasan.pdf` next to `index.html` — the hero "Resume" button already points at it.
